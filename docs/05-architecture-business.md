# 05 — Бізнес-архітектура

Цей документ — для нетехнічного читача, який хоче розуміти **які зовнішні сервіси ми використовуємо, скільки що коштує, де знаходяться гроші**.

---

## Зовнішні сервіси (постачальники)

### 1. OpenAI — мізки і голос реального часу

**Що дає:** найрозумніші мовні моделі і двосторонній голосовий канал.

| Модель | Для чого |
|---|---|
| `gpt-realtime-translate` | Realtime tab — однонаправлений переклад «голос → голос» |
| `gpt-realtime` | Chat tab — двосторонній «розумний перекладач» з системним промтом |
| `gpt-4o-transcribe` | Розпізнавання мови всередині `gpt-realtime` |

**Тарифікація:** OpenAI бере **за час сесії** (хвилини аудіо input + output). Це дорого — приблизно $0.06-0.30 за хвилину. Тому у нас є cost guard з автозупинкою.

**Ключове:** ми не зберігаємо аудіо у себе. WebRTC-з'єднання йде безпосередньо браузер ↔ OpenAI або iOS ↔ OpenAI. Наш сервер тільки видає короткостроковий ephemeral token (`ek_...`) і відразу про це забуває.

### 2. Soniox — швидке розпізнавання голосу в реальному часі

**Що дає:** розпізнавання голосу (STT) у форматі streaming — повертає текст рядок за рядком, поки людина ще говорить.

**Модель:** `stt-rt-v4` — найновіша real-time, підтримує одночасну роботу з 60+ мовами і визначає мову автоматично.

**Тарифікація:** значно дешевша за OpenAI realtime. Платимо за час аудіо. Ідеально для Voice tab де треба швидкі partial transcripts.

### 3. ElevenLabs — синтез голосу

**Що дає:** синтез людського голосу різними мовами (TTS).

**Модель:** `eleven_turbo_v2_5` — швидка, з низькою затримкою, підтримує 32+ мови включно з українською.

**Як використовуємо:**
- Voice tab: окремий батчевий запит — текст переходить → MP3.
- (У Mini App Live tab був streaming через WebSocket — у iOS поки не зробили).

**Тарифікація:** за кількість символів в озвученому тексті. Дешево.

Також у ElevenLabs є **Scribe v2** STT — використовуємо у iOS Voice tab v1 для одношотної транскрипції (фолбек, поки live Soniox не розгорнули в усіх режимах).

### 4. Groq — миттєвий переклад тексту

**Що дає:** чат-моделі OpenAI-сумісного API, але крутяться на спеціальних чіпах LPU. Latency ~50 мс — це ~10× швидше за стандартний GPT.

**Модель:** `gpt-oss-120b` (open-source 120B GPT від OpenAI, хоститься на Groq).

**Як використовуємо:** живий debounced переклад тексту в Voice tab. Поки користувач говорить, кожні 350 мс відправляємо накопичений текст на переклад, і встигаємо отримати відповідь до того, як юзер додасть нове слово.

**Тарифікація:** за токени input + output, дешево, 0 latency tax.

**Альтернатива:** Google Gemini (`gemini-3-flash-preview`) — є в коді як fallback (`TRANSLATION_PROVIDER=gemini`). Якісніший, але повільніший.

### 5. Apple — авторизація на iOS

**Що дає:** Sign in with Apple, дозволяє юзеру не реєструватись окремо.

**Тарифікація:** безкоштовно (включено в Apple Developer Program $99/рік).

### 6. Hetzner — наш сервер

**Що дає:** VPS у Гельсінкі — фізично крутиться вся наша backend-інфраструктура.

- IP: `89.167.19.222`
- 8 ГБ RAM, 75 ГБ SSD
- Тарифікація: ~€20/міс

### 7. MongoDB — наша база даних

**Що дає:** зберігає юзерів, їхні налаштування, статистику використання токенів.

**Хостинг:** Docker контейнер на Hetzner-сервері, окремо керованих сервісів типу Atlas не використовуємо. Бекап — поки відкрите питання.

### 8. Telegram — точка входу для бота і Mini App

**Тарифікація:** безкоштовно. Telegram нам нічого не платить, але і не бере.

## Як проходить грошовий потік (цикл одного перекладу)

### Voice tab — одношотний переклад через iOS

```
Юзер тапає мікрофон
  ↓
[Soniox WS] стрімить PCM 16kHz                       ← платимо Soniox $/min
  ↓ (текст рядок за рядком)
[Groq /api/translate-auto] переклад                   ← платимо Groq $/token
  ↓
Юзер тапає Stop, потім «🔊 Speak»
  ↓
[ElevenLabs /api/tts] синтез голосу                   ← платимо ElevenLabs $/char
  ↓
MP3 → AVAudioPlayer
```

### Realtime tab — синхронний переклад

```
Юзер тапає мікрофон
  ↓
[Сервер Teycan /api/realtime/session] (1 раз)         ← безкоштовно зі сторони Teycan
  ↓ (повертає ephemeral client_secret)
[iOS ↔ OpenAI WebRTC] весь час сесії                  ← платимо OpenAI $/min
  ↓
Авто-стоп через 3 хв АБО юзер тапає Stop
```

### Chat tab — двосторонній переклад

Той самий потік як Realtime, тільки інший endpoint OpenAI (`/v1/realtime/calls?model=gpt-realtime`) і повна session config (voice, instructions, VAD) надходить через наш сервер при міненні токена.

## Звідки беруться гроші (TBD)

Поки що **продукт некомерційний** — використовується автором особисто і в alpha-тесті серед знайомих. Усі витрати покриваються з кишені автора.

Можливі напрямки монетизації (відкриті):

1. **Freemium на iOS:**
   - Free: 30 хв Realtime + Chat на місяць, безлімітний Voice (через дешеві Soniox + Groq + ElevenLabs).
   - Premium $X/міс: 5 годин Realtime + Chat, кращі голоси, кастомні промти.

2. **In-app пакети** — купити 1 годину Realtime за $Y.

3. **B2B API** — продавати compliance-friendly API для готелів, лікарень, юридичних фірм.

4. **Підписка на Telegram Premium-feature** — після поглинання, але Telegram тут платежі складно.

5. **Корпоративні підписки** — для компаній що працюють з мігрантами / іноземними клієнтами.

Поки не вирішено. Залежить від того, чи буде продукт мати traction після публічного запуску.

## Місячні витрати (приблизно, на одного активного юзера)

Припускаючи юзер активно користується ~30 хв на день у Realtime/Chat + 50 одношотних перекладів:

| Сервіс | Витрата | $/місяць |
|---|---|---|
| OpenAI Realtime (15 годин) | ~$30 | **$30** |
| Soniox + Groq + ElevenLabs (Voice/Live) | ~$5 | **$5** |
| Hetzner (фіксовано на всіх) | ~$0.50 | **$0.50** |
| **Разом на юзера на місяць** | | **~$35** |

Тому **subscription мав би бути від $50/міс** при такому use pattern. Або обмежувати безкоштовний tier лімітами часу.

Cost guard захищає нас від «юзер залишив сесію відкритою на ніч» — без нього один такий юзер міг би з'їсти $300 за вечір.
